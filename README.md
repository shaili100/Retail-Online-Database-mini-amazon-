# Retail-Online-Database-mini-amazon-
# SQL Project

This project demonstrates a relational database design for a retail store and complex SQL JOIN queries to fetch complete order, product, and payment details.

## 🛠 Tools Used
- MySQL
- MySQL Workbench
- SQL

## 🗂 Database Tables
- users
- orders
- order_items
- products
- categories
- payments

## 🔗 Key SQL Concepts Used
- INNER JOIN
- Table Aliasing
- Foreign Keys
- Multi-table joins
- Error debugging (Error 1054)

## 📊 Sample Query Output
The main query retrieves:
- User name
- Order ID
- Product name
- Category
- Quantity
- Payment method & status

## ▶ How to Run
1. Run `schema.sql`
2. Run `sample_data.sql`
3. Run queries from `queries.sql`
